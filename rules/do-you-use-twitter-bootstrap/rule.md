<p>Efficient programmers do not re-invent the wheel. That's why we <a href="/_layouts/15/FIXUPREDIRECT.ASPX?WebId=3dfc0e07-e23a-4cbb-aac2-e778b71166a2&amp;TermSetId=07da3ddf-0924-4cd2-a6d4-a4809ae20160&amp;TermId=5af9a4e4-31fc-405a-942e-401e74ea347a">use the best Web UI libraries</a>.</p>
<p>Twitter Bootstrap is a NuGet Package that provides a jump-start for HTML based projects. It includes the HTML, CSS and JavaScript framework used by Twitter, to build the Twitter site.</p>
<br><excerpt class='endintro'></excerpt><br>
