<p>​Knowing the context and use of certain strings will help translators choose the right translation from the beginning when you localize your web application to make it available in a varitety of languages. Most translation tools will allow translators to see these comments as they translate the strings.&#160;To make your&#160;<span style="line-height&#58;20.7999992370605px;">comments&#160;</span>more friendly, it's better to list in the comment field where this text is used. e.g.<br></p><ul class="p1"><li><span style="line-height&#58;1.6;">​Text</span></li><li><span style="line-height&#58;1.6;">Validation<br></span></li><li><span style="line-height&#58;1.6;">Tooltip<br></span></li><li><span style="line-height&#58;1.6;">Textbox Placeholder​​</span></li></ul>
