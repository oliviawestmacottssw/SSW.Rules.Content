Events should end in "ing" or "ed".


<!--endintro-->

public event Action
&lt; connectioninformation &gt; ConnectionProblem;


::: bad
Bad code

:::


public event Action
&lt; connectioninformation &gt; ConnectionProblemDetected;


::: good
Good code
:::
